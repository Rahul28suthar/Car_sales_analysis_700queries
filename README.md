🚗 Car Sales Data Analysis using SQL

Project Overview

This repository contains 700 SQL queries used to analyze a car sales dataset.
The analysis focuses on identifying patterns in vehicle sales, customer income distribution, regional performance, and pricing trends.

The project demonstrates SQL-based analysis on structured data with queries ranging from basic data retrieval to advanced analytical queries.

Dataset Information

The dataset contains records of car purchases and customer attributes.
main feilds:
| Column        | Description                            |
| ------------- | -------------------------------------- |
| Car_id        | Unique identifier for each car sale    |
| Date          | Date of the transaction                |
| Customer Name | Name of the customer                   |
| Gender        | Gender of the customer                 |
| Annual Income | Annual income of the customer          |
| Dealer_Name   | Name of the car dealer                 |
| Company       | Car manufacturer                       |
| Model         | Model of the vehicle                   |
| Engine        | Engine specification                   |
| Transmission  | Transmission type (Manual / Automatic) |
| Color         | Color of the vehicle                   |
| Price ($)     | Price of the car                       |
| Dealer_No     | Dealer identification number           |
| Body Style    | Vehicle body type (SUV, Sedan, etc.)   |
| Phone         | Customer contact number                |
| Dealer_Region | Region where the car was sold          |
